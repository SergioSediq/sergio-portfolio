# Keycap Logo Checklist - Verify All Logos Are Correct

## How to Add/Update Logos in Spline

1. **Select the keycap object** in the Objects panel
2. In the **Properties panel**, find the **Material** section
3. Look for **Texture** or **Image** slot
4. Click on it and choose **"Load Image from URL"** or **"Import Image"**
5. Paste the icon URL from the list below
6. Adjust the material settings if needed

## How to Verify Logos Are Correct

1. **In Spline:**
   - Select each keycap
   - Check the Material section
   - Verify the texture/image matches the skill

2. **In Browser:**
   - Hover over each keycap
   - The hover message should match the skill
   - Visually check the logo looks correct

3. **If Logo is Missing:**
   - Select the keycap in Spline
   - Add the icon URL from the list above
   - Save the file
   - Refresh the browser

---

## Notes

- Some skills (VHDL, ITIL) don't have standard icons, so we're using placeholder icons
- If an icon URL doesn't work, try downloading the icon and importing it as a file
- Make sure to save the Spline file after adding logos!
